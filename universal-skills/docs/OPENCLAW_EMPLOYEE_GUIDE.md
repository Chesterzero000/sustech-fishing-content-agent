## 🤖 OpenClaw 是什么？

OpenClaw 是你的**全天候 AI 员工**，通过飞书与你交互，自主完成各种任务。它不是简单的聊天机器人，而是一个能够：
- 🔍 **主动监控**系统健康状态
- ✅ **自动审核**用户发布的内容

- 🚀 **自动部署**新功能和更新
- 🔧 **智能修复**常见问题

## 💼 OpenClaw 可以做什么？

### 1. 系统监控员（monitor-agent）

**工作内容**：
- 每 5 分钟检查一次系统健康状态
- 监控 API 响应时间、错误率
- 检查数据库、Redis、MCP 服务器状态
- 监控容器运行状态

**主动告警**：
- API 错误率 > 5% → 立即飞书告警
- 响应时间 > 2 秒 → 飞书告警
- 任何服务宕机 → 立即飞书告警
- 磁盘空间 < 10% → 飞书告警

**你可以问它**：
```
/status                    # 查询当前系统状态
/health check             # 执行健康检查
/metrics api              # 查看 API 指标
/alerts list              # 查看告警历史
```

---

### 2. 内容审核员（review-agent）

**工作内容**：
- 自动审核用户发布的帖子和评论
- AI 分析内容风险（违规、广告、敏感信息）
- 低风险内容自动通过（90% 的内容）
- 高风险内容推送给你人工复核

**审核规则**：
- **自动通过**：风险分数 < 0.3，无敏感词
- **自动拒绝**：风险分数 > 0.8，包含违法内容
- **人工复核**：风险分数 0.3-0.8，包含 1-5 个敏感词

**你可以问它**：
```
/review queue             # 查看待审核队列
/review retry <post_id>   # 重新审核某条内容
/review stats             # 查看审核统计
/review rules             # 查看审核规则
```

**它会主动推送**：
- 高风险内容需要你复核
- 附带 AI 分析结果和建议
- 你可以直接在飞书点击"通过"或"拒绝"

---

### 3. 数据分析师（report-agent）

**工作内容**：
- 每天早上 8:00 自动生成每日报告
- 每周一早上 9:00 生成每周摘要
- 分析用户增长、内容趋势、热门钓点

**每日报告包含**：
1. **用户数据**
   - 新增用户数
   - 活跃用户数
   - 留存率

2. **内容数据**
   - 新增帖子数
   - 新增评论数
   - 热门帖子 Top 10

3. **渔获数据**
   - 新增渔获记录
   - 热门钓点
   - 热门鱼种

4. **系统健康**
   - API 调用量
   - 错误率
   - 平均响应时间

**你可以问它**：
```
/report daily             # 手动触发每日报告
/report weekly            # 手动触发每周报告
/stats today              # 查看今日数据
/stats users              # 查看用户统计
/stats posts              # 查看内容统计
/trending spots           # 查看热门钓点
```

---

### 4. 技能验证员（skill-verify-agent）

**工作内容**：
- 验证新增的 Skill 代码
- 在沙箱环境中测试
- 自动部署通过验证的 Skill

**验证流程**：
1. **静态检查**：语法、安全、依赖
2. **沙箱测试**：隔离环境执行
3. **AI 审核**：代码质量评估
4. **自动部署**：通过后自动上线

**你可以问它**：
```
/skill verify <skill_name>    # 验证某个 Skill
/skill deploy <skill_name>    # 部署某个 Skill
/skill list                   # 查看所有 Skills
/skill status <skill_name>    # 查看 Skill 状态
```
