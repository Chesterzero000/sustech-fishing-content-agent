# 内容生产流程

## 目标

每天半自动生成一条适合南科大钓鱼协会发布的钓鱼知识科普内容。

## 推荐流程

1. 选题生成
   - 输入季节、天气、水域、鱼种或活动主题。
   - 输出 3-5 个候选选题。

2. 资料检查
   - 从 `data/` 知识库里读取鱼种、钓法、装备和注意事项。
   - 避免生成错误或危险建议。

3. 正文生成
   - 生成短文、长文或图文脚本。
   - 保持适合社团公众号、朋友圈、小红书或飞书群发布的语气。

4. 审核
   - 检查事实、表达、风险提示和是否适合公开发布。

5. 导出
   - 输出 Markdown 或纯文本。
   - 保存到 `output/`。

## 建议内容类型

- 鱼种科普
- 钓法入门
- 装备选择
- 季节水情
- 安全提醒
- 校园周边钓点观察
- 协会活动预热和复盘

## 发布边界

- 不公开敏感钓点细节。
- 不鼓励违规捕捞或危险水域活动。
- 涉及户外安全时必须提醒结伴、穿救生装备和遵守当地规定。
