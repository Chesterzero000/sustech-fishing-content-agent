# 仓库重命名方案

## 命名信息

- 当前 GitHub 仓库名：`fishing-agent-robot`
- 中文项目名：`南科大钓鱼协会内容机器人`
- 建议英文 slug：`sustech-fishing-content-agent`
- 建议完整仓库地址：`Chesterzero000/sustech-fishing-content-agent`

## 为什么建议重命名

`fishing-agent-robot` 能表达“钓鱼 + agent”，但没有体现南科大钓鱼协会和内容生产属性。`sustech-fishing-content-agent` 更适合作为长期项目名。

## GitHub 页面操作步骤

1. 打开仓库页面：`https://github.com/Chesterzero000/fishing-agent-robot`
2. 进入 `Settings`
3. 在 `General` 页面找到 `Repository name`
4. 将名称改为：`sustech-fishing-content-agent`
5. 点击 `Rename`

## 重命名后需要检查

- README 中的项目说明是否仍然准确。
- 后续脚本、自动化任务或外部链接是否使用旧仓库地址。
- 本地 clone 的 remote 是否更新。

## 本地仓库 remote 更新命令

```bash
git remote set-url origin https://github.com/Chesterzero000/sustech-fishing-content-agent.git
git remote -v
```
