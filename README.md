# 南科大钓鱼协会内容机器人

这是一个面向南科大钓鱼协会的自动化内容创作项目，目标是利用 AI Agent、MCP 工具和 Skills，半自动生成每日钓鱼知识科普内容。

## 当前状态

当前仓库只有 README，尚未加入实际代码、skills、选题库、素材或自动化脚本。

建议下一步把项目拆成以下模块：

- `data/`：鱼种、钓法、装备、季节、水域等知识库
- `scripts/`：内容生成、检查、导出脚本
- `output/`：每日生成内容样例
